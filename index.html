Previous Projects

Watch Your Garden Grow

Three week project created on a team of 7 at We Can Code IT. Developed through Agile methodologies and pair programming using SCRUM. Users are able to enter their zip code to return plants available for their plant habitable zone using an AJAX call, returning JSON objects. The users are able to see more details on each plant, displayed in a modal window. Plants can be added and removed from the desired plan, which is then generated with the appropriate planting date for their zip code.

My role focused entirely on the front end. The main view is designed using a responsive layout for the returned objects from the AJAX call, so that no matter how many or little plants were returned for the zip code, the view was easy to navigate. The layout was designed using CSS3 and DOM manipulation through Vanilla JavaScript without the assistance of any frameworks. At the end of each 3-day sprint, the front end was able to present a minimal viable product using the features designed from the back end. The back end was created through Java, JPA relational mapping, Spring MVC, and rest controllers to fuel the front end.

Music Artists Reviews

Music Reviews was a project involving the creation of a website through SpringMVC, using TDD, Java and the creation of a database with JPA. The front was designed in Thymeleaf, using Flexbox and Grid in CSS while applying AJAX and Javascript. The data for the views is accessed through a Spring Controller. Musical artists are `organized by genre. Users are able to create comments to be displayed on each musical artist's page. Artists were also organized by tags which users are able to create and delete from each artist's page using AJAX. Each tag will direct user to all artists associated with that tag.

This project was created in 3 weekend iterations as a project assignment for We Can Code IT.

Virtual Medieval Lair Amok

Upgraded version of Virtual Medieval Lair. Written in Java by TDD. Implemented interfaces and extended classes to create different abilities for different types of creatures. Virtual Medieval Lair

One of the first Java programs I created. Written in Java by TDD. User participated in a virtual pet game taking care of creatures at The Lair through the console. The creatures were stored in a HashMap via name as the key. Players are able to care for the different creatures, which effect different levels simultaneously. At the end of each round, a tick effect was applied effecting all levels of each creature. Users are able to add and remove creatures.
