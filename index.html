Previous Projects


Music Artists Reviews

Music Reviews was a project involving the creation of a website through SpringMVC, using TDD, Java and the creation of a database with JPA. The front was designed in Thymeleaf, using Flexbox and Grid in CSS while applying AJAX and Javascript. The data for the views is accessed through a Spring Controller. Musical artists are `organized by genre. Users are able to create comments to be displayed on each musical artist's page. Artists were also organized by tags which users are able to create and delete from each artist's page using AJAX. Each tag will direct user to all artists associated with that tag.

This project was created in 3 weekend iterations as a project assignment for We Can Code IT.


Virtual Medieval Lair Amok

Upgraded version of Virtual Medieval Lair. Written in Java with TDD. Implemented interfaces and extended classes to create different abilities for different types of creatures using polymorphism. Game started with an already-programmed list of creatures and allowed the game player to add or remove creature. Introduced new levels that the game player needed to help control.


Watch Your Garden Grow

Three week project created on a team of 7 at We Can Code IT. Developed through SCRUM methodologies and pair programming.


Virtual Medieval Lair

Previous version of Virtual Medieval Lair Amok. Written in Java with TDD. User participated in a game taking care of creatures at The Lair. The creatures were stored in a HashMap via name as the key. Players were able to care for the different creatures, which effect different levels simultaneously. Starting levels were encapsulated. Game players were able to add and remove creatures.
